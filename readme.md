# Выше крыши — МТС True Tech Cup 2026

Предсказание высоты зданий Санкт-Петербурга по двум источникам геоданных
с помощью Random Forest.

## Логика решения

1. **Очистка данных** — фильтрация дубликатов, невалидных геометрий и выбросов
   по площади в обоих источниках (A и B) независимо.
2. **Нормализация адресов** — приведение адресных строк к единому формату
   (без сокращений «г.», «ул.», «д.» и т.д.).
3. **Матчинг A+B** — пространственное сопоставление зданий:
   - spatial join по пересечению полигонов
   - расчёт IoU, cover_a/b, centroid distance
   - построение графа совпадений и поиск компонент связности
   - уточнение сложных кластеров (1×N, N×1, N×N)
   - выбор репрезентативной геометрии и агрегация атрибутов
4. **Фич-инжиниринг** — 45 признаков: геометрические (площадь, периметр,
   NPI, convexity), пространственный контекст (KDE, статистики соседей в радиусе 100 м),
   категориальные (этажность, назначение, источник высоты).
5. **Обучение Random Forest** — логарифмическая целевая переменная,
   аугментация высоких зданий (>50 м), sample weights по диапазонам высот.
6. **Инференс** — предсказание высот для всех зданий без разметки + визуализация.

## Структура проекта
├── pipeline.py                         # точка входа — запускает все шаги
├── clean_source_a.py                   # очистка Источника A
├── clean_source_b.py                   # очистка Источника B
├── normalize_addr_a.py                 # нормализация адресов A
├── normalize_addr_b.py                 # нормализация адресов B
├── matching.py                         # матчинг A+B → финальный датасет
├── rf_train.py                         # обучение модели
├── predict_and_visualize.py            # инференс + графики
└── README.md

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

### Полный пайплайн (очистка → матчинг → обучение → предсказание)

```bash
python pipeline.py src_A.csv src_B.csv ./output
```

Результаты окажутся в `./output/`:
- `final_buildings_with_addr_and_new_features.csv` — финальный датасет
- `rf_final_model.joblib` + `rf_final_model.json` — модель и метаданные
- `predictions/predictions_all_buildings.csv` — предсказанные высоты

### Только инференс (модель уже обучена)

```bash
python predict.py \
  output/final_buildings_with_addr_and_new_features.csv \
  output/rf_final_model.joblib \
  output/rf_final_model.json \
  predictions/predictions_new.csv \
  plots/
```
